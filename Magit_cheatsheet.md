# Magit cheatsheet

C-x g -- Magit / Git status

## Pulling

F -- display menu
p to pull

## Staging / unstaging

s -- to stage changes
u -- to unstage changes

## To commit changes

c -- commit menu
c again -- to commit

enter commit message

C-c C-c to actually commit changes

## Push up changes

P -- display push menu 
p -- to push
